# 在线教学与实训平台 - 部署文档
## 1. 环境要求
### 1.1 服务器要求
操作系统：windows 10/11

CPU：2核心及以上

内存：4GB及以上（推荐8GB）

硬盘：20GB及以上（用于存储上传文件和数据库）

网络：公网IP或域名，开放80/443端口（HTTP/HTTPS）

### 1.2 软件依赖
|软件|	版本|	说明|
|---|---|---|
|Python	|3.10+	|后端运行环境|
|Node.js|	18.x / 20.x|	前端构建环境|
|MySQL|	5.7 / 8.0|	数据库|
|Redis|	6.x / 7.x|	Celery 消息代理（暂未强制使用）|
|Nginx|	1.18+|	反向代理与静态文件服务|
|Git|	最新版|	代码拉取|
|pip|	最新版|	Python包管理|
|npm / yarn|	最新版|	前端包管理|

### 2.2 安装 Node.js (如果未安装)

## 3. 后端部署
### 3.1 创建 Python 虚拟环境
```bash
cd web_backend
python3 -m venv venv
source venv/bin/activate
```
### 3.2 安装 Python 依赖
创建 requirements.txt（若没有，可手动创建并包含以下内容）：

```text
Django==5.0.6
djangorestframework==3.15.1
django-cors-headers==4.3.1
mysqlclient==2.2.4
celery==5.3.6
redis==5.0.1
docker==7.0.0
pymysql==1.1.0
```
安装：

```bash
pip install -r requirements.txt
```
### 3.3 配置数据库
登录 MySQL 创建数据库和用户：

```sql
CREATE DATABASE online_teach CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'teach_user'@'localhost' IDENTIFIED BY 'your_strong_password';
GRANT ALL PRIVILEGES ON online_teach.* TO 'teach_user'@'localhost';
FLUSH PRIVILEGES;
```
修改 web_backend/web_backend/settings.py 中的数据库配置：

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'online_teach',
        'USER': 'teach_user',
        'PASSWORD': 'your_strong_password',
        'HOST': 'localhost',
        'PORT': '3306',
    }
}
```
### 3.4 配置静态文件和媒体文件
在 settings.py 中设置：

```python
STATIC_URL = '/static/'
STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')

MEDIA_URL = '/media/'
MEDIA_ROOT = os.path.join(BASE_DIR, 'media')
```
### 3.5 配置跨域和允许的主机
```python
ALLOWED_HOSTS = ['your-domain.com', 'www.your-domain.com', '服务器IP']
CORS_ALLOWED_ORIGINS = [
    'https://your-domain.com',
    'http://localhost:5173',  # 开发时可保留
]
```
### 3.6 生成密钥与迁移
```bash
# 生成新的 SECRET_KEY (替换 settings.py 中的默认值)
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"

# 执行数据库迁移
python manage.py makemigrations
python manage.py migrate

# 创建超级管理员（可选，用于后台管理）
python manage.py createsuperuser
```

### 3.7 测试运行
```bash
python manage.py runserver 0.0.0.0:8000
通过浏览器访问 http://服务器IP:8000 验证后端接口（如 /api/courses/ 应返回401，表示需要认证）。
```
## 4. 前端部署（构建）
### 4.1 安装依赖
```bash
cd ../web_frontend   # 进入前端目录
npm install
```

修改 src/api/client.js 中的 baseURL：

```javascript
const api = axios.create({
```
