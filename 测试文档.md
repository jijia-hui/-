# 在线教学与实训平台 - 测试文档
## 1. 测试概述
### 1.1 测试目标
验证平台所有功能符合需求规格，确保用户能够顺利完成课程浏览、选课、作业管理、提交与评分等核心业务流程，并保证系统的稳定性、安全性和易用性。

### 1.2 测试范围
前端界面：所有页面渲染、交互、表单验证、响应式布局。

后端 API：认证、课程、作业、提交、评分等接口的正确性、权限控制与数据一致性。

文件上传：参考文档上传、学生提交文件上传（文本格式）。

权限控制：学生/教师角色的不同权限隔离。

异常处理：网络错误、表单校验、文件格式错误等场景。

### 1.3 测试环境
环境	配置
操作系统	Windows 11 (客户端)
浏览器	Chrome 120+,  Edge 120+
后端	Django 5.0.6 + MySQL 8.0 + Gunicorn + Nginx
前端	React 18 + Vite 5 + Ant Design 5
测试工具	Jest (前端单元测试)
